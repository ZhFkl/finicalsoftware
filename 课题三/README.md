# Graph Cube backend

One designed graph cube backend project.

### Install

```
$ pip install fastapi
$ pip install "uvicorn[standard]"
```

### Setup
```
$ uvicorn main:app --reload
```

Open your browser at [http://127.0.0.1:8000/](http://127.0.0.1:8000).
